# Blog Featured Image Prompts

## Image Strategy

### Technical Specs

- **Dimensions:** 1920x1080 (16:9) for featured images
- **Format:** Generate at high res, convert to WebP at 80% quality
- **Size Target:** < 200KB per image
- **Naming:** `{slug}-featured.webp`

### Aesthetic Requirements (from Design Manifesto)

- **Background:** Void Navy (#0B0E14) - dark, not pure black
- **Accent:** Cyber Lime (#CCF381) - use for highlights, glows, focal points
- **Style:** Neo-brutalist, technical, server room aesthetic
- **Mood:** "High-precision instrument found in a dark server room"
- **Avoid:** Generic AI art, soft gradients, bright colors, rounded elements

### Prompt Structure

```
[Subject], [Style modifiers], dark moody lighting, void navy background #0B0E14,
cyber lime accent highlights #CCF381, technical industrial aesthetic,
neo-brutalist design, sharp edges, high contrast, professional photography, 8k
```

## Conversion Commands

After generating images with Flux/Nano Banana Pro:

```bash
# Single image conversion
cwebp -q 80 input.png -o output.webp

# Batch conversion with resizing
for f in *.png; do
  convert "$f" -resize 1920x1080 -quality 80 "${f%.png}.webp"
done

# Check final sizes
ls -lh *.webp
```
